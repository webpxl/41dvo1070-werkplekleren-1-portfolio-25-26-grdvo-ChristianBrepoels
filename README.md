# Portfolio – Christian Brepoels

Dit is mijn persoonlijk portfolio dat ik heb gemaakt voor het vak **Werkplekleren (WPL1)**.

Op deze website stel ik mezelf voor, toon ik mijn vaardigheden en laat ik mijn schoolprojecten en eigen projecten zien. Het portfolio geeft een overzicht van wie ik ben en wat ik kan op vlak van digitale vormgeving en webontwikkeling.

---

## Gebruikte technologieën

Voor dit portfolio heb ik uitsluitend gebruikgemaakt van:
- HTML
- CSS
- JavaScript (plain JavaScript, zonder frameworks)

De website is volledig responsive en werkt op desktop, tablet en smartphone.

---

## Gebruik van AI-tools

Tijdens het maken van dit portfolio heb ik gebruikgemaakt van AI-tools (zoals ChatGPT) om:
- hulp te krijgen bij het schrijven en verbeteren van teksten
- technische problemen in HTML, CSS en JavaScript op te lossen
- structuur en logica in mijn code te verbeteren

Alle code en inhoud heb ik zelf aangepast, getest en begrepen.

---

## Bronnen en materiaal

- Alle afbeeldingen van projecten zijn eigen werk en gemaakt voor schoolopdrachten of persoonlijke projecten.
- De gebruikte iconen en afbeeldingen worden enkel gebruikt binnen dit portfolio.
- Externe links (zoals LinkedIn en Instagram) verwijzen naar mijn eigen profielen.
